# :potable_water: NEM Faucet

## :fire: Setup

```shell
# boot NIS on local
$ ./nixNis.sh

# install packages
$ npm install
$ bundle install

# prepare database
$ bundle exec hanami db prepare

# start app
$ bundle exec hanami server
```
